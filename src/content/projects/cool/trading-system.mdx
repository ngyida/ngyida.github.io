## Summary

This project is about learning how to write C++ that is safe, maintainable, and scalable.

In building an MVP-scoped trading system, I focus on things like design patterns, code modularity, repository structure, and writing cleaner C++.
There is still a lot left to explore, especially around testing, error handling, and the less glamorous parts that make systems actually hold up.
