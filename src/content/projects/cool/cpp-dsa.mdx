## Summary

This project is about learning how to write C++ with more control, more speed, and a better understanding of the machine underneath.

By building data structures with strict performance requirements, I get to focus on things like memory behavior, object semantics, resource management, and STL usage.
Still a long way to go, but that is part of the point. I am thinking of pushing toward compile-time and hardware-level optimization next.
