## context

This was one of the most research-oriented environments I worked in. The core problem space was advanced, but the contribution still needed to translate into something usable by collaborators and external evaluators.

## what I worked on

My work included:

- contributing to a homomorphic encryption compiler effort
- building Python bindings so the system became easier to test and integrate
- experimenting with machine-learning use cases on encrypted data to probe feasibility

## what mattered technically

The Python binding work stood out because it was a translation problem in the best sense. It required preserving performance and interface clarity while exposing a system that would otherwise remain distant from many users.

## what I learned

I learned that I enjoy research work most when it has an implementation bridge. I like the moment where advanced ideas have to become usable surfaces rather than remaining purely conceptual.
