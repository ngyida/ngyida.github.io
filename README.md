# Yi Da Portfolio

An editorial, content-first portfolio built with Astro and MDX.

## Development

- `npm run dev`
- `npm run build`
- `npm run preview`
- `npm run deploy`

## Content Authoring

- Edit the about page in `src/content/pages/about.mdx`
- Edit the career overview intro in `src/content/pages/career.mdx`
- Edit the projects overview intro in `src/content/pages/projects.mdx`
- Edit the contact page in `src/content/pages/contact.mdx`
- Add a new career deep dive by creating a new file in `src/content/career/`
- Add a new project deep dive by creating a new file in `src/content/projects/`

The site structure, routing, and styling live in Astro layouts, helpers, and pages. The written content lives in MDX files so you can expand the site without touching the layout code.
