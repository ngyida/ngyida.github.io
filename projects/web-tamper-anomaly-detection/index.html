Explainable Anomaly Detection

2024 - 2025

Making a new framework usable

Context

Webnorm is a newly proposed anomaly detection framework for web tampering. In plain terms, the goal was to identify when a website had been modified in suspicious ways, while keeping the detection process more structured and reliable than a simple bag of heuristics.

The research project was developed through a collaboration between researchers from GovTech, NUS, and SJTU.

My work

Webnorm had promising initial results, but it also had some fairly obvious next questions around usability, robustness, and cost.

I focused on three areas:

  • improving the explainability of its results
  • studying security concerns around the use of LLM components in the system
  • reducing the cost of using the tool in practice

In short, the work was about making the tool viable in an industrial context i.e. real companies can actually use it. The longer version of this project lives in my dissertation, if you want the full write-up.
