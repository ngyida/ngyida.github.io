Sensitive Chat App

2022

repo

Summary

As a chronic bad texter, I have the habit of skimping through messages and end up missing important nuances.

Thus, I built a messaging web application that integrated emotional recognition and sentiment analysis into chat messages to help me recognize important messages without context.

This was done in 2022 using ML models, and it is now obsolete thanks to ChatGPT…
